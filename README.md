--------------------------------------------------------------
Installation
--------------------------------------------------------------

To install (only on Ubuntu) follow these simple instructions:

1. 

```
cd ~
```

2.

```
git clone git@github.com:devGunnin/.dotfiles
```

3.

```
cd .dotfiles
```

4.

```
bash install
```

5.

```
cd ~/.config/nvim/
```

6.

```
vim .
```

7. In Neovim 

```
:PackerInstall
```

8. In Neovim

```
:PackerSync
```

9. Restart terminal, and follow instructions on the screen.
